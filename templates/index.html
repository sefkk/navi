Welcome to Navi! You can try different path-finding algorithms with different travel modes and priorities.

Some remarks are: Not all roads are walkable, so walking mode may not always find a path. Same goes with public transport.

The map may sometimes contain isolated nodes which can never find a path.

Just press "Generate Map" and start the journey, to try different variations on the same map, you can just reset path and calculate a new path.

Enjoy!
